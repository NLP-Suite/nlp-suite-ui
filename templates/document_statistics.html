In inputDirectory the script expects a single txt file or a directory containing a set of txt files.
In outputDirectory, the script generates a set of csv files each containing word n-grams between 1 and 3.
When n-grams are computed by sentence index, the sentence displayed in output is always the first occurring sentence.
Please, tick the checkbox if you wish to compute basic statistics on your corpus.
Use the dropdown menus to lemmatize words or exclude stopwords/punctuation and select the desired option.
Please, tick the checkbox if you wish to compute statistics on your corpus by POS (Part of Speech) tag (e.g., nouns, verbs, adjectives).
The scripts relies on the Stanza POS annotator to export all tokens in the corpus with their POS value.
In outputDirectory, the script generates a set of csv files each containing the words in our corpus by POS value and their frequencies. Charts will also be visualized.
Select the type of data transformation you wish to apply in plotting charts. All charts involving multiple documents are automatically normalized by document size.
