Enter the number of topics to be used. Try different number of topics for better results
Tick/untick the checkbox to run the BERtopic LLM topic modeling. BERTopic was developed by Maarten Grootendorst in 2022. BERTopic is based on BERT, released by Google in October 2018.
Tick/untick the checkbox to run the java-based MALLET LDA(Latent Dirichlet Allocation) topic modeling algorithm, first released in 2002. MALLET was developed by Andrew McCallum of the University of Massachusetts Amherst. MALLET does not come with any visual displays of its own.
Tick/untick the checkbox to run the Gensim LDA(Latent Dirichlet Allocation) topic modeling algorithm. Gensim uses the MALLET algorithm and displays the results in beautiful interactive visuals.
Charts can be visualized automatically as bar, line, or pie charts. Select your preferred option. Many more chart types are available in the specialized GUIs data_visualization_1_main.py and data_visualization_2_main.py. Open those GUIs and select the csv file and variable you wish to chart.
